# Shop Platform

A local business discovery and promotion platform that helps users find the right nearby shops quickly and helps shop owners grow their customer base through visibility and promotion.

---

# 🚀 Vision

Build a scalable platform where:

* Customers can search and discover shops easily
* Shop owners can promote their businesses
* Businesses can increase visibility and customer reach
* The platform can later support promotions, analytics, premium listings, and advertisements

---

# 🎯 Core Features

## Customer Side

* Search nearby shops
* Search by category and location
* View shop details
* Discover top shops

## Shop Owner Side

* Register shop
* Manage dashboard
* Promote listings
* View customer engagement

# 📌 Development Rules

* No inline CSS
* Separate CSS and JS files
* Modular code structure
* Reusable components
* Clean folder organization
* Small focused commits
* Proper naming conventions

---

# 🔥 Git Commit Style

```text
feat: add search page
fix: dashboard layout issue
style: improve navbar design
```

---

# 🧠 Team Workflow

* Do not push directly to main branch
* Use feature branches
* Review code before merging
* Keep code modular and readable
