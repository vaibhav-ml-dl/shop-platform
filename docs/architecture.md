# Shop Platform Architecture

## Project Goal

Build a scalable local business discovery and promotion platform.

The platform helps:
- customers discover nearby shops
- shop owners promote their businesses
- businesses gain visibility and customers

---

# Core Principles

- Modular architecture
- Reusable components
- Scalable structure
- Clean codebase
- Easy collaboration
- Feature isolation

---

# Frontend Structure

Frontend contains:
- pages
- css
- js
- assets

Rules:
- No inline CSS
- No huge JS files
- Separate logic and UI
- Reusable components

---

# Backend Structure

Backend contains:
- routes
- models
- database logic

Rules:
- Separate APIs by feature
- No huge app.py
- Database logic isolated
- Clean API responses

---

# Development Workflow

1. Create feature branch
2. Build small isolated feature
3. Test locally
4. Commit clean code
5. Push to GitHub
6. Merge after review

---

# Git Rules

Commit examples:

feat: add homepage layout
fix: navbar responsive issue
style: improve dashboard UI

---
